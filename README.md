# Supportik
Описание:
Утилита, написанная на языке программирования Python, которая должна облегчить работу всем, кто работает с доменами, сайтами и не только. Недохакерам тоже буду рад.
Description:
A utility written in the Python programming language, which should make the work of everyone who works with domains, websites and more easier.

Утилита иногда будет обновляться, постепенно увеличивая количество возможностей. Для того, чтобы узнать список команд и получить пояснение к ним, введите в утилите команду "help". Приложен журнал изменений в версиях в файле changelog.
